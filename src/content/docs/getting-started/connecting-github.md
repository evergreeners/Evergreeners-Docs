# Connecting GitHub

Evergreeners relies on GitHub data to track your contributions and growth.

## Why Connect GitHub?

Connecting your GitHub account allows Evergreeners to:
- Sync your contribution history.
- Automatically update your streak.
- Verify quest completions.
- Display your "Garden" based on real code activity.

## How to Connect

1. Go to your **Profile Settings**.
2. Click on the **Connect GitHub** button.
3. Authorize the Evergreeners application on GitHub.
4. Select the repositories you want to track (optional).

## Permissions

We only request access to public activity and contribution data. We do not require write access to your private repositories unless explicitly requested for specific features.
