# Contributing to Evergreeners

We love contributors! Whether you're fixing bugs, adding features, or improving documentation, your help is welcome.

## Ways to Contribute

1. **Code**: Check the [GitHub issues](https://github.com/evergreeners/evergreeners/issues) for tasks labeled `help wanted`.
2. **Docs**: Help us improve this documentation by submitting PRs to the `docs` folder.
3. **Design**: Suggest UI improvements or create new assets for the Quest system.
4. **Community**: Help answer questions in the Discord or GitHub discussions.

## Development Workflow

See our [Getting Started](/getting-started/installation) guide for how to set up the project locally.

## Pull Request Guidelines

- Create a descriptive title.
- Reference the issue you're solving.
- Include screenshots for UI changes.
- Ensure your code passes all linting and tests.
